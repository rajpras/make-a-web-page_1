The Box model



Content is the very center - what is being displayed. Like the nucleus
Padding is around the content and is following whatever is in the background
Border is around the padding like the edge of paper
Margin is transparent and the space behind it is inivible like white space

To get dimensions of an element width would be left border, padding, content, right padding and right border


Box sizing creates a border and box - which is border and padding. Does not include margin
For Older browsers there are browser specific prefixes
Add this information to the top of the CSS file
/*CSS - *{
-webkit-Box-sizing: border-box; Width: 460px
-moz-Box-sizing: border-box; Width: 460px
-ms-Box-sizing: border-box; Width: 460px
Box-sizing: border-box;
Width: 460px}
*/
Boxes can also be the %
To get a outlines around all divs(used for debugging) - /*
